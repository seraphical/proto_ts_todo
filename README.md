# 前端

## 思路分析

事件层: 用户点击触发事件, 然后更新数据,并且更新视图
数据层: 点击后更新数据
视图层: 更新数据之后对视图进行更新

- 模版层, 每次更新视图都是对模版的一次操作

### dom 创建

dom 创建有两种方式

1. 改变数据, dom 只用根据 data 来生成然后渲染就行
2. 根据 data 精确生成 dom, 精确删除 dom,改变 dom

### 初始化

第一种方案初始化就直接开头创建一次
第二种方案进行初始化就遍历初始数据创建来 appendChild

### 优劣分析

第一种方案, 整个 dom 重新进行改变和生成, 过于冗余
第二种方案类似于虚拟 dom

哪个简单先用哪个吧
